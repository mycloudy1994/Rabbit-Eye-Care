# Privacy Policy

Last updated: 2026-05-03

Rabbit Eye Care - Free Break Reminder respects your privacy.

## What this extension does

Rabbit Eye Care - Free Break Reminder provides timed eye-care break reminders inside Chrome. It can show notifications, display a break page, and place an overlay on webpages during break time.

## Data we collect

Rabbit Eye Care - Free Break Reminder does not collect personal data such as your name, email address, payment information, contacts, or browsing history for transmission to the developer.

## Data stored locally

The extension stores the following settings locally in your browser:

- break interval
- break duration
- sound on/off
- night mode preference
- timer state needed for reminders to continue working

This data is stored only on your device through Chrome extension storage.

## Data sharing

Rabbit Eye Care - Free Break Reminder does not sell, share, or transfer your personal data to third parties.

## Remote code and servers

Rabbit Eye Care - Free Break Reminder does not load remote code for its core functionality.

## Permissions usage

- `storage`: saves your reminder settings locally
- `alarms`: schedules reminder and break timers
- `notifications`: shows break reminders and heads-up notifications
- `tabs`: sends overlay messages to open tabs during break time
- `offscreen`: plays reminder sounds
- `host_permissions <all_urls>`: allows the extension to show the break overlay on normal webpages you open in Chrome

## Contact

If you need support or have privacy questions, please contact the developer through the support email listed on the Chrome Web Store page.
